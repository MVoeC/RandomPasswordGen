# 随机密码生成器

## 项目简介

这是一个基于 Python Flask 和 HTML/CSS/JavaScript 的随机密码生成工具。该工具提供了美观的 Web 界面，可以根据用户自定义的规则生成安全的随机密码。

### 主要特性

- ✨ **美观的用户界面**：采用现代化设计，渐变色背景，响应式布局
- 🔐 **高度可定制**：支持自定义密码长度、数量和字符类型
- 🎯 **灵活的字符选择**：
  - 小写字母 (a-z)
  - 大写字母 (A-Z)
  - 数字 (0-9)
  - 特殊字符 (!@#$%^&*等)
  - 自定义字符支持
- 🚫 **字符排除功能**：可排除容易混淆的字符（如0O、1lI等）
- 📊 **密码强度指示**：实时显示生成密码的强度评估
- 📋 **一键复制**：方便快捷地复制所有生成的密码
- 🔒 **安全可靠**：使用 Python 的 random 模块生成真随机密码

### 技术栈

- **后端**：Python 3.x + Flask
- **前端**：HTML5 + CSS3 + JavaScript (原生)
- **样式**：CSS3 渐变、Flexbox、Grid 布局

## 安装与运行

### 环境要求

- Python 3.7 或更高版本
- pip 包管理器

### 安装步骤

1. **克隆或下载项目到本地**

2. **安装依赖**

```bash
pip install -r requirements.txt
```

### 运行项目

```bash
python app.py
```

服务器启动后，在浏览器中访问：
```
http://localhost:5000
```

## 使用说明

### 基本使用

1. **设置密码长度**：输入所需的密码长度（4-128位，建议12位以上）
2. **设置密码数量**：输入要生成的密码数量（1-100个）
3. **选择字符类型**：勾选想要包含的字符类型（至少选择一种）
   - 小写字母
   - 大写字母
   - 数字
   - 特殊字符
4. **自定义字符**（可选）：添加任何自定义字符到字符池
5. **排除字符**（可选）：输入想要排除的字符，如 `0Oo1Il`
6. **点击"生成密码"按钮**
7. **查看结果**：生成的密码会显示在下方列表中
8. **复制密码**：点击"复制全部"按钮一键复制所有密码

### 高级功能

#### 排除易混淆字符

在"排除字符"输入框中输入容易混淆的字符，例如：
- `0Oo` - 排除数字0和字母O
- `1lI` - 排除数字1、小写L和大写i
- `5S` - 排除数字5和大写S

#### 自定义字符集

在"自定义字符"输入框中可以添加任何特殊字符，例如：
- 中文字符：`你好世界`
- Emoji：`😀😃😄`
- 其他符号：`±§¶`

## API 接口

### POST /generate

生成随机密码的 API 接口

**请求体**（JSON）：
```json
{
  "length": 16,
  "count": 5,
  "lowercase": true,
  "uppercase": true,
  "digits": true,
  "special": true,
  "customChars": "",
  "excludeChars": "0Oo1Il"
}
```

**响应**（JSON）：
```json
{
  "success": true,
  "passwords": [
    "aB3d#Fg9Hj2K@mN",
    "pQ5r$St7Uv8W%xY",
    "zA2b&Cd4Ef6G*hI",
    "jK8l^Mn9Pq3R#sT",
    "uV6w@Xy7Za5B$cD"
  ]
}
```

## 安全说明

- 本工具使用 Python 的 `random` 模块生成密码，适用于一般用途
- 对于需要更高安全性的场景，建议使用 `secrets` 模块
- 生成的密码不会被存储在服务器上
- 建议在本地运行使用，避免通过不安全的网络传输密码

## 功能特点详解

### 密码强度评估

系统会根据密码长度自动评估强度：
- **弱**：少于8位
- **中等**：8-11位
- **强**：12位及以上

### 字符池管理

程序会根据用户选择的选项和自定义字符构建字符池，然后从中排除指定的字符，确保生成的密码符合所有要求。

### 错误处理

- 输入验证：检查密码长度和数量的有效性
- 字符池检查：确保排除字符后仍有可用字符
- 友好的错误提示：当出现问题时显示清晰的错误信息
