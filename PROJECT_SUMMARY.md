## 🔒 安全说明

- ✅ 使用 Python random 模块生成随机密码
- ✅ 密码不会存储在服务器
- ✅ 所有计算在服务器端完成
- ⚠️ 建议在本地环境使用
- ⚠️ 如需更高安全性，可替换为 secrets 模块

## 🌐 API 接口

### POST /generate

**请求示例：**
```json
{
  "length": 16,
  "count": 5,
  "lowercase": true,
  "uppercase": true,
  "digits": true,
  "special": true,
  "customChars": "",
  "excludeChars": "0Oo1Il"
}
```

**响应示例：**
```json
{
  "success": true,
  "passwords": [
    "aB3d#Fg9Hj2K@mN",
    "pQ5r$St7Uv8W%xY",
    "zA2b&Cd4Ef6G*hI"
  ]
}
```
