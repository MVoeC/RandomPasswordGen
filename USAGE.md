# 快速使用指南

## 启动项目

### 1. 安装依赖（首次运行）
```bash
pip install -r requirements.txt
```

### 2. 启动服务器
```bash
python app.py
```

### 3. 访问应用
打开浏览器访问：http://localhost:5000

## 功能说明

### 基本操作
1. **密码长度**：设置生成密码的位数（4-128位）
2. **密码数量**：设置一次生成多少个密码（1-100个）
3. **选择字符类型**：勾选想要包含的字符类型
   - ☑ 小写字母 (a-z)
   - ☑ 大写字母 (A-Z)
   - ☑ 数字 (0-9)
   - ☑ 特殊字符 (!@#$%^&*等)

### 高级功能

#### 自定义字符
在"自定义字符"框中输入任何你想添加的字符，例如：
- 中文：你好世界
- Emoji：😀😃😄
- 其他：±§¶

#### 排除字符
在"排除字符"框中输入想要排除的字符，常见例子：
- `0Oo` - 排除数字0和字母O（容易混淆）
- `1lI` - 排除数字1、小写L和大写i
- `5S` - 排除数字5和大写S
- `2Z` - 排除数字2和大写Z

### 使用技巧

1. **安全建议**：
   - 密码长度建议 ≥ 12位
   - 至少包含3种字符类型
   - 定期更换密码

2. **排除易混淆字符**：
   - 输入 `0Oo1lI5S2Z` 可排除大部分容易混淆的字符
   - 适用于需要手动输入密码的场景

3. **一键复制**：
   - 生成密码后点击"复制全部"按钮
   - 所有密码会以换行分隔复制到剪贴板

## 常见问题

**Q: 为什么点击"生成密码"没反应？**
A: 请确保至少选择一种字符类型，或输入自定义字符。

**Q: 可以生成纯数字密码吗？**
A: 可以，只勾选"数字"选项即可。

**Q: 如何生成只包含字母和数字的密码？**
A: 取消勾选"特殊字符"选项，只保留字母和数字选项。

**Q: 排除字符后显示"字符池为空"？**
A: 这说明你排除的字符太多，导致没有可用字符了。请减少排除的字符。

## 安全提示

⚠️ **重要**：
- 生成的密码不会保存在服务器
- 建议在本地环境运行使用
- 不要在不安全的网络环境下使用
- 生成后请妥善保管密码
